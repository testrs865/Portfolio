# 株分析ツール (Python)

株価データを取得し、テクニカル指標（移動平均・RSI）と売買シグナル（ゴールデンクロス / デッドクロス）を可視化するPythonアプリケーションです。

---

## 概要

本プロジェクトは、株価データの分析・可視化を通じて、データアナリティクスの基礎スキル（データ取得・前処理・特徴量生成・可視化）を実践的に示すことを目的としています。

* Yahoo Finance(Yahoo! Finance（米国版）から金融データを取得するための、Pythonのオープンソースライブラリ)
* 移動平均線によるトレンド分析
* RSIによる過熱感の判定
* 売買シグナルの自動検出
* Matplotlibによる可視化

---

## 使用技術

* Python 3.11
* yfinance
* pandas
* matplotlib

---

## 機能一覧

### 1. 株価データ取得

指定した銘柄コードと期間から株価データを取得

### 2. テクニカル指標

* 5日移動平均（MA5）
* 25日移動平均（MA25）
* RSI（14日）

### 3. 売買シグナル検出

* ゴールデンクロス（買い）
* デッドクロス（売り）

### 4. 可視化

* 株価 + 移動平均線
* 売買シグナルのプロット
* 出来高
* RSI（別パネル表示）

## 使い方

### 1. リポジトリをクローン

```bash
git clone https://github.com/testrs865/Learning.git
```

### 2. 必要ライブラリをインストール

```bash
pip install yfinance matplotlib pandas
```

### 3. 実行

```bash
python main.py
```

### 4. 入力

```
銘柄コードを入力してください（例：4755.T）
開始日を入力してください（例：2023-01-01）
終了日を入力してください（例：2024-01-01）
```
